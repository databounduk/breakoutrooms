# Break Out Rooms Streamlit App

Welcome to the breakout rooms application.

Looking into the developer forum we can see that Zoom are looking at providing an API in the future to enable this to be carried out programatically.

## Installation

This application requires python 3.8 install and the install of the requirements file provided

```bash
pip install -r requirements.txt
```

## Usage

Once the requirements are install, the application can be run using the following command at the root directory

```bash
streamlit run app.py
```
